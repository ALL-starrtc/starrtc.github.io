其他配套组件

StarPlayer 视频播放器

StarPlayer是为SDK配套开发的视频播放组件，用来展示视频画面，具体使用方法参见demo。

API说明

//让player在窗口顶层，会挡住所在窗口的其它内容
void setZOrderOnTop(boolean b);

//让player在其他player之上，但还是在窗口的底层
void setZOrderMediaOverlay(boolean b);

//设置player的显示模式
void setScalType(StarPlayerScaleType type);

StarWhitePanel 白板 & StarWhitePanel2 白板2

StarWhitePanel是为SDK配套开发的教育白板类组件，可以实时直播手指在屏幕上划出的轨迹,具体使用方法参见demo。

API说明

//开始对外广播（直播者调用）
void publish();

//停止对外广播（直播者调用）
void pause();

//填充轨迹数据
void setPaintData(byte[] data,String fromID);

//清空历史轨迹
void clean();

网络IO及视频编码参数日志

added in SDK v1.1.8

试用视频相关业务时，可以通过以下方式获取相关信息
网络相关信息
网络质量(0~1)：StarRtcCore.currentNetworkQuality;
上传速度(kb/s)：StarRtcCore.currentUploadSpeed;
下载速度(kb/s)：StarRtcCore.currentDownloadSpeed;

视频大图信息
分辨率：StarRtcCore.currentVideoWidth，StarRtcCore.currentVideoHeight;
码率(kbps)：StarRtcCore.currentBitRate;
帧率(fps)：StarRtcCore.currentFPS;

视频小图信息
分辨率：StarRtcCore.currentVideoWidthSmall.currentVideoHeightSmall;
码率(kbps)：StarRtcCore.currentBitRateSmall;
帧率(fps)：StarRtcCore.currentFPSSmall;
